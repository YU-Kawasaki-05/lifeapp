# Codex Assets

このディレクトリは、Codex 運用関連の参照資料をまとめた場所です。

## 含まれるもの
- `CODEX_SETUP.md`: このリポで適用したセットアップ内容
- `GUIDE_ja.md`: Codex 構成ガイド
- `SOURCES.md`: ベストプラクティスの参照元一覧
- `COMMUNITY_SOURCES.md`: 非公式ソースの採用メモ
- `CLAUDE_COMMANDS_COMPAT.md`: `.claude/commands` 互換運用（Codex prompts化）
- `ENVIRONMENT_USAGE_GUIDE.md`: このリポでの運用手順（実行コマンド含む）
- `codex-template-kit/`: テンプレート一式のアーカイブ

## 実運用の設定場所
- `AGENTS.md` (repo root)
- `.codex/`
- `.pre-commit-config.yaml`
- `.githooks/`
- `.agents/skills/`（canonical）
- `skills/`（`.agents/skills/` への互換 symlink）
- `codex/rules/`
