## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| [ジャーニーマップ]/[時間分析] | タブ切替 | 2つのビュー切替 |
| 階層ツリー | ツリービュー | ノードタップで展開/折り畳み。タスクタップでSCR-32 |
| プログレスバー | バー | 配下の完了率から自動算出 |
| 🤖 AIコンテキスト | カード | 現在のタスクと上位目標の接続を説明 |
| 積み上げタイムライン | 縦タイムライン | 過去の完了マイルストーンを時系列表示 |
| 円グラフ/棒グラフ | チャート | PJ別の時間配分。タップで詳細 |
| 理想vs実績 | 比較バー | 乖離がある場合に色で強調（赤: 大きな乖離） |
| トレンドグラフ | 折れ線グラフ | 月別推移。PJ別に色分け |
| [理想の時間配分を設定] | ボタン | 各PJの理想%を設定するモーダル表示 |
| [+ 目標を追加] | ボタン | 長期ゴール or 中期目標の追加フォーム |
