## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| 統計カード | 数値表示 | 登録数、アクティブ率をリアルタイム集計 |
| システム状態 | ステータス表示 | 各外部APIとDBの接続状態。異常時は🔴 |
| ユーザーテーブル | テーブル | ソート・検索・フィルタ可能 |
| (停止) | ボタン | 確認ダイアログ → アカウント停止 |
| (復帰) | ボタン | 停止アカウントを復帰 |
| ページネーション | ナビ | 1ページ20件表示 |
