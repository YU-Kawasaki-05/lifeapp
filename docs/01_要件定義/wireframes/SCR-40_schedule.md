## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| [AIでスケジュール生成] | ボタン | AIがActive PJ + GCal + 生活リズムからタイムボクシング生成。確認ダイアログ後に反映 |
| [GCal pull ↓] | ボタン | Google Calendarから最新予定を取得。成功/失敗をトースト通知 |
| [GCal push ↑] | ボタン | ARDORSのタイムブロックをGoogle Calendarに一括反映。確認ダイアログあり |
| タイムブロック | ドラッグ可能な矩形 | タップで詳細表示。ドラッグ&ドロップで時間変更。リサイズで長さ変更 |
| 📅 GCal予定 | 矩形（色分け） | ARDORSブロックとは別色で表示。編集不可（GCal側で編集） |
| 🚶 移動 | ブロック | GCal予定の場所情報から自動生成（FR-34、P2） |
| 🌿 探索タイム | ブロック（特別色） | AIが自動組込。ユーザーが削除しようとすると「探索時間は大切です」と確認 |
| 現在時刻ライン | 水平線 | 現在時刻を示す赤い横線 |
| (日) / (週) | タブ切替 | 日表示 / 週表示の切り替え |
