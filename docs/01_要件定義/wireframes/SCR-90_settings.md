## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| AI厳しさレベル | ラジオボタン | 3択。選択変更は即時保存 |
| 通知ON/OFF | トグルスイッチ | 各通知タイプの有効/無効 |
| 時刻/曜日選択 | ドロップダウン | 通知のタイミング設定 |
| 生活リズム | ドロップダウン | AIのスケジュール生成に使用 |
| GCal連携状態 | ステータス表示 | 接続中/未接続。接続ボタンでOAuth開始 |
| 時間配分の理想値 | 数値入力 + バー | 各PJの理想%。合計が100%でない場合に警告 |
| (習慣の管理画面を開く) | リンク | SCR-50に遷移 |
| [ログアウト] | ボタン | 確認ダイアログ → セッション破棄 → SCR-03 |
