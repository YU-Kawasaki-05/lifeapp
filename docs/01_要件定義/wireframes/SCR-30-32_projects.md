## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| PJカード（SCR-30） | カード | タップでSCR-31に遷移 |
| [+ 新規PJ作成] | ボタン | PJ作成フォーム表示（名前、カテゴリ、ゴール、期限入力） |
| Active/Warm/Cold セクション | セクション | PJの状態別にグループ表示 |
| (( 🟢🟡🔴 )) | ヘルススコアバッジ | 自動算出。タップで算出根拠表示 |
| タスク行（SCR-31） | リスト行 | タップでSCR-32（タスク詳細）をモーダル表示 |
| ☐ / ☑ チェック | チェックボックス | タスク完了/未完了のトグル |
| [+ タスク追加] | ボタン | インラインでタスク名入力 → 作成 |
| コンテキスト復元 | 通知カード | 最終作業から3日以上経過した場合に自動表示 |
| (ゴールマップで見る →) | リンク | SCR-70に遷移（このPJの目標にフォーカス） |
