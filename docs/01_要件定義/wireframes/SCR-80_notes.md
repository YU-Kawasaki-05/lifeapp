## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| [+ 新しいメモ] | ボタン | 新規メモ作成モーダルを表示 |
| {検索} | テキスト入力 | インクリメンタルサーチ |
| フィルタタグ | トグルボタン群 | 複数選択可。選択中のタグのメモのみ表示 |
| メモカード | カード | タップでインライン展開 or モーダルで詳細表示 |
| 🏷 タグ | バッジ | メモのカテゴリを示す |
| 関連PJ | リンク | タップでSCR-31（PJ詳細）に遷移 |
| [🎤 音声入力] | ボタン | マイクON → STT → テキストに変換 |
| AI関連提示 | カード | セレンディピティ・エンジン（P2）によるメモ間の関連 |
