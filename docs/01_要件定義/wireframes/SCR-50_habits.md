## 習慣作成/編集フォーム（モーダル）

```
+----------------------------------------------+
| 新しい習慣を作る                        (×)  |
+----------------------------------------------+
|                                              |
|  習慣名                                      |
|  {朝ランニング                            }  |
|                                              |
|  きっかけ（いつ・何をしたら？）               |
|  {起床後すぐ                              }  |
|                                              |
|  最小行動（2分以内で完了できるレベル）         |
|  {靴を履いて外に出る                      }  |
|  💡 ハードルを下げるのがコツ。                |
|     「走る」ではなく「靴を履く」から。         |
|                                              |
|  if-then plan                                |
|  もし {起床後すぐ        } なら              |
|     → {靴を履いて外に出る}                   |
|                                              |
|  頻度                                        |
|  (●毎日) (○平日のみ) (○週N回) (○カスタム)   |
|                                              |
|  プロジェクト紐付け（任意）                   |
|  {ランニング習慣         v}                  |
|                                              |
|              [保存]  (キャンセル)             |
|                                              |
+----------------------------------------------+
```

## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| [+ 新しい習慣] | ボタン | 習慣作成モーダルを表示 |
| (編集) | リンク | 習慣編集モーダルを表示 |
| (削除) | リンク | 確認ダイアログ → 削除 |
| ヒートマップ | カレンダー形式 | ■=実行済 □=未実行。GitHubのcontributionグラフ風 |
| ストリーク | テキスト | 現在の連続日数 + 最長記録 |
| 💡 ガイド | ヒントテキスト | 最小行動のコツを表示 |
| if-then plan | 構造化入力 | 「もし{X}なら→{Y}」のテンプレート |
