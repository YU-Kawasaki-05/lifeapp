## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| AIブリーフィング | カード（自動表示） | 1日1回、初回アクセス時。閉じるボタンで非表示可 |
| エネルギーチェックイン | 1タップウィジェット | 気分(3段階)/体力(5段階)/集中(5段階)を各1タップ |
| タイムライン | スケジュール表示 | ブロックタップで詳細。SCR-40への導線 |
| 習慣チェック | チェックボックス | 1タップで完了記録。ストリーク自動更新 |
| [デイリークローズを始める] | CTAボタン | SCR-60 レビュー（日次モード）に遷移 |
| PJカード | カード | タップでSCR-31（PJ詳細）に遷移 |
| ((🤖)) | フローティングボタン | タップでSCR-21F（AIパネル）を開く |
| 時間帯切替 | 自動 | ユーザー設定の生活リズムに基づいて時間帯を判定 |
