## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| [日次]/[週次]/[月次] | タブ切替 | モード切替。URLのクエリパラメータも更新 |
| 📊 サマリー/レポート | 自動生成カード | AIがデータから自動生成。AI障害時はデータのみ表示 |
| まとめて入力エリア | テキストエリア + 音声ボタン | ユーザーが自由に振り返り・意向を入力 |
| AI分析結果 | カード | 送信後にAIが構造化・分析した結果を表示 |
| 対話エリア | チャット形式 | AI分析後に深掘り対話 |
| PJ状態変更（週次） | ドロップダウン | Active/Warm/Coldを変更 |
| 週次ゴール設定 | 編集可能リスト | AIが3つ提案。ユーザーが修正/追加/削除 |
| [AIでスケジュール生成]（週次） | ボタン | 週次ゴール + GCal予定からタイムボクシング生成 |
| [承認してGCalに反映]（週次） | CTAボタン | スケジュール確定 → Google Calendar push |
| 目標見直し（月次） | 編集可能リスト | 長期ゴール・中期目標の追加/修正/削除 |
| Future Self Letter（月次） | テキストエリア | 3ヶ月後に届く手紙を作成 |
| [レビューを完了] | CTAボタン | レビュー結果を保存 → SCR-20に遷移 |
