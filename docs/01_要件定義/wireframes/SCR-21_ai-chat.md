## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| チャット履歴 | スクロール領域 | 過去の対話を時系列表示 |
| {メッセージ入力} | テキスト入力 | Enter or 送信ボタンでAIに送信 |
| [🎤 音声] | ボタン | マイクON → STT → テキストに変換 → 確認後送信 |
| [送信 >>>] | ボタン | メッセージをAIに送信 |
| ☑ タスク候補 | チェックボックス+テキスト | AIが提案したタスク。個別に承認/却下 |
| [PJを作成] / [メモだけ残す] | アクションボタン | AI提案に対するユーザーの選択 |
| [スケジュールを調整] | アクションボタン | 承認 → SCR-40に反映 |
| (×) | 閉じるボタン | フローティングパネルを閉じる |
| (↗) | 全画面表示ボタン | SCR-21（全画面チャット）に遷移 |
| ((🤖)) | フローティングボタン | 閉じた状態からパネルを再度開く |
