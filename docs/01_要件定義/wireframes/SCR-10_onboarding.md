## 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| (スキップ) | テキストリンク | オンボーディングをスキップ → SCR-20 |
| {自由入力} | テキストエリア | 複数行入力可。プレースホルダーに例文 |
| [🎤 音声入力] | ボタン | マイクON → STT → テキストエリアに反映 |
| [次へ] | CTAボタン | 次のStepに遷移 |
| (戻る) | リンク | 前のStepに戻る |
| ☑ / ☐ | チェックボックス | AI構造化結果の個別承認/却下 |
| (編集) | リンク | インラインで内容を編集可能にする |
| [ARDORSを始める] | CTAボタン | 承認済み項目をデータ化 → SCR-20 |
