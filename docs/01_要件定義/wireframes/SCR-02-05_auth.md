```
+================================================================+
| <<<ロゴ>>> ARDORS                                               |
+================================================================+
|                                                                 |
|                    ARDORSを始める                                |
|                                                                 |
|              +-----------------------------+                    |
|              |                             |                    |
|              |  [G] Googleで登録            |                    |
|              |                             |                    |
|              |  ──── または ────            |                    |
|              |                             |                    |
|              |  メールアドレス              |                    |
|              |  {email@example.com       } |                    |
|              |                             |                    |
|              |  パスワード                  |                    |

|              |                             |                    |

|              |  (ログインはこちら)           |                    |
|              |                             |                    |
|              +-----------------------------+                    |
|                                                                 |
+================================================================+
```

### スマホ レイアウト

```
+-----------------------------+
| <<<ロゴ>>> ARDORS           |
+-----------------------------+
|                             |
|     ARDORSを始める           |
|                             |
| [G] Googleで登録             |
|                             |
| ──── または ────             |
|                             |
| メールアドレス               |
| {email@example.com       }  |
|                             |
| パスワード                   |
| {••••••••        } (目)     |
| ████░░░░ 普通                |
| 8文字以上、英字+数字を含む    |
|                             |
| [登録する]                   |
|                             |
| (ログインはこちら)            |
|                             |
+-----------------------------+
```

### 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| [G] Googleで登録 | OAuthボタン | Google認証画面（外部）に遷移 |
| {email} | テキスト入力 | メール形式バリデーション |
| {password} | パスワード入力 | 入力に応じて強度バーが変化 |
| (目) | トグルアイコン | パスワードの表示/非表示切替 |
| ████░░░░ | 強度バー | 弱(赤)/普通(黄)/強(緑) |
| [登録する] | CTAボタン | 確認メール送信 → SCR-10 |
| (ログインはこちら) | リンク | SCR-03に遷移 |

## SCR-03: ログイン

### PC レイアウト

```
+================================================================+
| <<<ロゴ>>> ARDORS                                               |
+================================================================+
|                                                                 |
|                    ログイン                                      |
|                                                                 |
|              +-----------------------------+                    |
|              |                             |                    |
|              |  [G] Googleでログイン        |                    |
|              |                             |                    |
|              |  ──── または ────            |                    |
|              |                             |                    |
|              |  メールアドレス              |                    |
|              |  {email@example.com       } |                    |
|              |                             |                    |
|              |  パスワード                  |                    |
|              |  {••••••••        } (目)    |                    |
|              |                             |                    |
|              |  (パスワードを忘れた方)       |                    |
|              |                             |                    |
|              |  [ログイン]                  |                    |
|              |                             |                    |
|              |  アカウントをお持ちでないですか？|                  |
|              |  (新規登録はこちら)           |                    |
|              |                             |                    |
|              +-----------------------------+                    |
|                                                                 |
+================================================================+
```

### 要素詳細

| 要素 | 種類 | 動作 |
|------|------|------|
| [G] Googleでログイン | OAuthボタン | Google認証 → SCR-20 or SCR-10 |
| [ログイン] | CTAボタン | 認証成功 → SCR-20 or SCR-10 |
| (パスワードを忘れた方) | リンク | SCR-04に遷移 |
| (新規登録はこちら) | リンク | SCR-02に遷移 |
