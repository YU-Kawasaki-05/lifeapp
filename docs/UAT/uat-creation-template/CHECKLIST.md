# 実行チェックリスト

## 事前準備

- [ ] Claude Code CLI がインストールされている
- [ ] `claude --version` コマンドが動作する
- [ ] このテンプレートディレクトリを作業場所にコピーした

## ファイル準備

### 基本パターン（設計書が整理済み）

- [ ] `01-inputs/業務一覧表.md` あるいはそれに類するファイルを配置（必須）
- [ ] その他の設計書をMarkdown形式で `01-inputs/` に配置
- [ ] `.claude/uat-config.json` をプロジェクトに合わせて編集（オプション）

### 自動選別パターン（docs/ ディレクトリから選別する場合）

- [ ] プロジェクトの `docs/` ディレクトリに設計書が存在することを確認
- [ ] Claude Code起動後、`/input-prepare` コマンドを実行
- [ ] `docs/` ディレクトリから自動的にファイルが分析・選別される（`docs/UAT/` は除外）
- [ ] `01-inputs/` に選別された設計書がコピーされたことを確認
- [ ] `.claude/uat-config.json` をプロジェクトに合わせて編集（オプション）

## 実行前確認

- [ ] 作業ディレクトリに移動した
- [ ] `01-inputs/` に必要なファイルが揃っている
- [ ] `業務一覧表.md` に対象業務が記載されている

## 実行

- [ ] `claude` コマンドで Claude Code を起動
- [ ] `/make-uat-complete` コマンドを実行
- [ ] （オプション）改善回数を指定: `/make-uat-complete 3`

## 実行中

- [ ] エラーが表示されていないか確認
- [ ] フェーズ1: スコープ定義書作成の完了を確認
- [ ] フェーズ2: 仮UAT仕様書作成の完了を確認
- [ ] フェーズ3: UAT仕様書検証の完了を確認
- [ ] フェーズ4: UAT仕様書改善の完了を確認

## 実行後確認

- [ ] `AI-generated/UATスコープ定義書.md` が作成されている
- [ ] `AI-generated/uat_specs/` にカテゴリ別UAT仕様書が作成されている
- [ ] `AI-generated/UAT仕様書_統合レポート.md` が作成されている
- [ ] 業務網羅率が目標値（通常100%）に達している

## 成果物レビュー

- [ ] 統合レポートで全体像を確認
- [ ] 各カテゴリのUAT仕様書を確認
- [ ] テストケースが網羅的であるか確認
- [ ] テストデータが適切であるか確認
- [ ] 期待結果が明確に記載されているか確認

## トラブル時

問題が発生した場合:

- [ ] Claude Code を再起動
- [ ] `/make-uat-complete` を再実行（途中から再開される）
- [ ] `README.md` のトラブルシューティングセクションを確認
- [ ] エラーログを確認（Claude Code のメッセージ）

## よくあるエラー

### 「対象業務が0件です」

- [ ] `.claude/uat-config.json` の `targetStatuses` を確認
- [ ] 業務一覧表のステータス列と設定が一致しているか確認

### 「権限エラー」

- [ ] `.claude/settings.local.json` の `permissions.allow` を確認
- [ ] 必要な権限が含まれているか確認

### 「処理が途中で止まる」

- [ ] Claude Code を再起動
- [ ] 同じコマンドを再実行（自動的に途中から再開）
- [ ] トークン制限に達していないか確認

## 完了

すべてのチェックが完了したら、成果物を活用してください。

- [ ] UAT仕様書を関係者に共有
- [ ] テストケースのレビューを依頼
- [ ] 必要に応じて手動で調整
- [ ] テスト実施計画に組み込む

---

**所要時間の目安**:
- 小規模プロジェクト（10-20業務）: 30分〜1時間
- 中規模プロジェクト（30-50業務）: 1〜1.5時間
- 大規模プロジェクト（70業務以上）: 1.5〜2時間

**トークン使用量の目安**:
- 1回実行: 約50,000〜80,000トークン
- 改善3回: 約150,000〜240,000トークン
- 改善5回: 約250,000〜400,000トークン
